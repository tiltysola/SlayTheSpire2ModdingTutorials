# 《杀戮尖塔2》模组制作指南

《杀戮尖塔2》原生支持模组，所以你可以直接制作模组并使用。

由于塔2刚发售暂时没有创意工坊，所以目前只能通过本地加载模组。下面介绍如何制作模组。

由于游戏目前处于抢先体验阶段，教程可能不是最优解，且代码更改频繁，如有错误请指正。
